# quiz_gemini
Descrição do Projeto

Este projeto é um Quiz interativo desenvolvido em Spring Boot, com integração à API do Google Gemini para auxiliar na geração de perguntas, respostas ou explicações adicionais dentro do sistema.

Tópicos:
	•	Desenvolvimento de aplicações web com Spring Boot
	•	Integração com APIs externas
	•	Organização de projeto backend
	•	Uso de serviços de IA generativa (Gemini)

⸻

Tecnologias Utilizadas
	•	Java 17+
	•	Spring Boot 3+
	•	Maven
	•	Google Gemini AI (google-ai-client)
	•	Thymeleaf (quando aplicável)
	•	HTML
